# Riverbed Data Store plugin for Grafana

**Riverbed Data Store plugin for Grafana** is an open‑source project developed by [Riverbed](https://www.riverbed.com) for the Grafana users in the [Riverbed Community](https://community.riverbed.com)

## Work in Progress

This project is currently under active development.

You can temporarily experiment with building the plugin yourself and using it in your own Grafana instance.

See the [How-to]](doc/How-to.md) for details.

Publishing the plugin to the official Grafana plugin catalog is also in progress.
